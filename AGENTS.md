# Instructions for LLM agents operating with the GoToSocial codebase

## Code Modification Rules

1. **No Automatic Changes** - You are **forbidden** from making any changes to the codebase on your own initiative.
2. **Strict Review Requirement** - All proposed changes must be presented to the user for **explicit approval** before any modifications are made.
3. **No Filesystem Operations** - You must not read, write, or delete any files unless explicitly authorized by the user for a specific task.
4. Read pixelfed_layer_implementation_plan.md before doing any changes to the codebase.
